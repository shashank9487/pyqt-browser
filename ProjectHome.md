# A simple PyQt Web Browser #
## system requirements ##
  * Qt >= 4.5
  * sip >= 4.8
  * Python >= 2.5
  * Python-pyqt >= 4.5

## features ##
  * Text Zoom-In, Zoom-Out, Zoom-Reset button
  * Provide a Bookmark menu (add and delete by a mouse click)
  * Whole page Zoom-In and Zoom-out by a mouse double click
  * Watch YouTube video
